## Conversor de Moeda

Projeto de conversor de moedas desenvolvido com HTML, CSS e JavaScript.
A aplicação permite converter valores entre diferentes moedas de forma simples, rápida e intuitiva.

## Funcionalidades

Conversão entre:

- Real (BRL)
- Dólar (USD)
- Euro (EUR)
- Libra (GBP)
- Bitcoin (BTC)

✔ Interface moderna e responsiva

✔ Atualização dinâmica das moedas

✔ Formatação automática de valores

## Tecnologias

 - HTML5
 - CSS3
 - JavaScript

## Aprendizados

Neste projeto pratiquei:

- Manipulação do DOM
- Eventos no JavaScript
- Condicionais
- Responsividade
- Formatação de moedas
